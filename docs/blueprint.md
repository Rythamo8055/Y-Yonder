# **App Name**: Yonder

## Core Features:

- Floating Dock Navigation: Provides a macOS-inspired floating dock for primary navigation, utilizing the Liquid Glass effect for a translucent, layered feel and the active navigation with clear indicators and fluid transitions.
- Content Carousels: Visually engaging carousels featuring full-bleed images or glass cards that peek from the edges, inviting horizontal swiping. Parallax effects add depth.
- Soft-Edged Buttons: Pill-shaped buttons that 'lift' upon being pressed, offering visual feedback through shadow and highlights.
- Bottom Sheet Modals: Modals emerging from the bottom of the screen that focus user attention with background blurring and the Liquid Glass effect.
- Newsfeed Summarizer: AI tool to generate short summaries of lengthy posts using the OpenAI models. This feature helps users quickly grasp the essence of shared articles or updates in their feed.

## Style Guidelines:

- Primary color: Rosewater (#dc8a78) for a modern and vibrant feel.
- Background color: Frappuccino (#eeecd9) provides a soft and airy base.
- Accent color: Mauve (#8839ef) is used for highlights and key interactive elements, enhancing focus.
- Headline font: 'Poppins', a geometric sans-serif, chosen for a precise and fashionable look for headlines.
- Body font: 'PT Sans', a humanist sans-serif, combines modernity with a touch of warmth, ensuring readability for body text.
- Custom icons with a thin, geometric style, mirroring the clean aesthetics of macOS.
- Bottom 'dock' style navigation bar, akin to macOS, is used for main navigation. Floating cards with blurred backgrounds create a sense of depth and hierarchy.
- Subtle physics-based animations throughout the UI that mimic Material Design, focusing on natural movements.
